One-Population Inference

Part 1

Confidence Intervals

Start: you want to estimate a population parameter
Goal is a range of plausible values + a confidence level
What are you estimating?
μ — population mean
Estimating a mean (μ)
The variable is quantitative
e.g. avg body temperature, avg folate intake, avg birth weight
p — population proportion
Estimating a proportion (p)
The variable is categorical (success/fail)
e.g. % vaccinated, % with antibodies, % who test positive
Are the CLT conditions met?
What to check before proceeding
Condition 1 — i.i.d. Random sample from a much larger population (or data generated by an i.i.d. process)
Condition 2a — sample size for μ n ≥ 20–30 is a rough rule of thumb. Larger n is always safer.
Condition 2b — sample size for p n·$\hat{p}$ > 10 and n·(1−$\hat{p}$) > 10
Use the sample proportion $\hat{p}$ here (not $p_0$).
✓ YES — conditions met
Proceed — CLT applies
Sampling distribution of the statistic is approximately Normal. The z-based CI formula is valid.
✗ NO — conditions not met
Stop — cannot use these formulas
Consult your instructor for an alternative method.
What confidence level do you need?
Find the critical value $z_{1-\alpha/2}$
Set $\alpha = 1 - $(confidence level as decimal), then compute $z_{1-\alpha/2} = $ qnorm(1 − α/2)

90% CI $\alpha = 0.10$
$z_{1-\alpha/2} = $ qnorm(0.95) ≈ 1.645
95% CI $\alpha = 0.05$
$z_{1-\alpha/2} = $ qnorm(0.975) ≈ 1.96
99% CI $\alpha = 0.01$
$z_{1-\alpha/2} = $ qnorm(0.995) ≈ 2.576
Which formula? (depends on the parameter)
μ — mean
CI for a mean (μ)
$$\bar{x} \pm z_{1-\alpha/2} \cdot \frac{s}{\sqrt{n}}$$ where: $\bar{x}$ = sample mean $s$ = sample standard deviation $n$ = sample size $z_{1-\alpha/2}$ = qnorm(1 − α/2) Width shrinks as $n$ ↑ or confidence ↓
p — proportion
CI for a proportion (p)
$$\hat{p} \pm z_{1-\alpha/2} \cdot \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}$$ where: $\hat{p}$ = sample proportion (# successes / $n$) $n$ = sample size $z_{1-\alpha/2}$ = qnorm(1 − α/2) $SE = \sqrt{\hat{p}(1-\hat{p})/n}$ is the standard error
Step 4 — Interpret in context
Template: "We are [100(1-α)]% confident that the population [mean / proportion] of [context] is between [lower] and [upper] [units]."
⚠ do NOT say "probability" Once data are collected and the interval is computed, the endpoints are fixed numbers — nothing is random anymore. Say "confident," not "there is a 95% chance."
What "confident" means The procedure has a 95% probability of capturing $\mu$ across many samples. Any specific interval either contains $\mu$ or it doesn't — we just don't know which.
Anatomy of any CI:   [point estimate] ± [margin of error]  =  [point estimate] ± $z_{1-\alpha/2}$ × [standard error]
The standard error (SE) estimates the SD of the sampling distribution using sample data. The margin of error = $z_{1-\alpha/2}$ × SE is half the width of the interval.
Part 2

Hypothesis Testing

Start: you want to test a claim about a population parameter
Is this one specific value plausible given the data we collected?
Step 1 — State the hypotheses
Null hypothesis $H_0$ The "assume-true" statement. Must contain equality (=, ≥, or ≤). This is what you assume is true and then may reject.
e.g. $H_0 : \mu = 98.6$  or  $H_0 : p = 0.5$
Alternative hypothesis $H_a$ What you're looking for evidence of. Together, $H_0$ and $H_a$ must cover all possible values of the parameter.
e.g. $H_a : \mu \neq 98.6$  or  $H_a : p < 0.5$
One-sided or two-sided test?
two-sided (Hₐ : ≠)
Two-sided test
You care whether the parameter is different in either direction from the null value.

$H_0 : \theta = \theta_0$    $H_a : \theta \neq \theta_0$

e.g. "Is the mean body temperature different from 98.6°F?"
"Is folate intake different from the recommended 100 μg?"
one-sided (Hₐ : < or >)
One-sided test
You care about one specific direction only — e.g. detecting harm, improvement, or a safety threshold.

Left-tailed: $H_0 : \theta \geq \theta_0$   $H_a : \theta < \theta_0$
Right-tailed: $H_0 : \theta \leq \theta_0$   $H_a : \theta > \theta_0$

e.g. "Is the HPV vaccination rate below 80%?"
"Is allele transmission rate above 50%?"
Step 2 — Are the CLT conditions met?
What to check (same i.i.d. + sample size rules — with one twist for proportions)
i.i.d. Random sample from a much larger population
Sample size for μ n ≥ 20–30 (rule of thumb)
Sample size for p — key difference! Use the null value $p_0$, not $\hat{p}$:
n·$p_0$ > 10 and n·(1−$p_0$) > 10
✓ YES — proceed
CLT applies — use z-test
Sampling distribution of the statistic is approximately Normal. The z-score test is valid.
✗ NO — stop
Stop — z-test invalid
Consult your instructor for an alternative method.
Step 3 — Compute the test statistic (z-score)
testing μ
Test statistic for μ
$$z = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}$$ where: $\bar{x}$ = sample mean $\mu_0$ = null hypothesis value $s$ = sample standard deviation $n$ = sample size
testing p
Test statistic for p
$$z = \frac{\hat{p} - p_0}{\sqrt{p_0(1-p_0) / n}}$$ where: $\hat{p}$ = sample proportion $p_0$ = null hypothesis value $n$ = sample size ⚠ Use $p_0$ in the denominator — NOT $\hat{p}$! One-sided null (e.g. $p \geq 0.8$): use the boundary value (0.8) as $p_0$.
Step 4 — Compute the p-value (depends on test direction)
two-sided
Two-sided p-value
p = 2·P(Z ≤ −|z|)

In R: 2 * pnorm(-abs(z))

Both tails count because we care about deviations in either direction. The null is a single point ($\theta = \theta_0$) so extreme values on either side count as evidence against it.
left-tailed (Hₐ: <)
Left-tailed p-value
p = P(Z ≤ z)

In R: pnorm(z)

Only the left tail counts — we're looking for evidence that $\theta$ is below the null boundary. Values larger than $z$ are not "more extreme" under this $H_a$.
right-tailed (Hₐ: >)
Right-tailed p-value
p = P(Z ≥ z)

In R: 1 - pnorm(z)

Only the right tail counts — we're looking for evidence that $\theta$ is above the null boundary.
Step 5 — Compare p-value to significance level α
Choosing α before collecting data
α is set in advance based on how serious a false alarm (Type I Error) would be.
Stringent ($\alpha = 0.01$)Pharmaceutical drug trials, medical devices — a false positive could harm patients
Standard ($\alpha = 0.05$)Default in most scientific research
Lenient ($\alpha = 0.10$)Low-stakes decisions — e.g. A/B testing a button color
p-value ≤ α
Reject $H_0$
Template: "At a significance level of α, we reject the null hypothesis. We have evidence to suggest that [$H_a$ in context]."

⚠ Possible Type I Error: rejecting $H_0$ when it is actually true. This happens $\alpha \times 100\%$ of the time even when $H_0$ is true.
p-value > α
Fail to reject $H_0$
Template: "At a significance level of α, we fail to reject the null hypothesis. We do not have evidence to suggest that [$H_a$ in context]."

⚠ Possible Type II Error: failing to reject $H_0$ when it is actually false. Mnemonic: "fail 2 reject = Type 2."
Never say "accept $H_0$" — failing to find evidence is not proof $H_0$ is true.

Error type reference

$H_0$ is actually TRUE $H_0$ is actually FALSE
Reject $H_0$ Type I Error
"False positive"
Probability = $\alpha$		 Correct decision ✓
Power = $1 - \beta$
Fail to reject $H_0$ Correct decision ✓ Type II Error
"False negative"
Probability = $\beta$
fail 2 reject = Type 2
Key principle behind hypothesis testing (proof by contradiction): Assume $H_0$ is true. Collect data. Ask: how probable is this data (or more extreme data) under that assumption? The p-value answers that question. If the answer is "very improbable" (p-value < $\alpha$), we take that as evidence against $H_0$ — just like a contradiction disproves an assumption in math.
